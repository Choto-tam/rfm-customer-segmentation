# rfm-customer-segmentation
Проект анализа транзакционных данных интернет-магазина с использованием **RFM-сегментации клиентов**.

# Описание проекта

Цель проекта — сегментировать клиентов на основе их покупательского поведения.

Используется метод **RFM (Recency, Frequency, Monetary)**:

* **Recency** — давность последней покупки

* **Frequency** — частота покупок

* **Monetary** — сумма покупок

Результат — выделение групп клиентов с разной ценностью для бизнеса.

# Бизнес-задача

Проект решает типичную задачу аналитики:

* определить наиболее ценных клиентов

* выявить клиентов с риском оттока

* сегментировать аудиторию для маркетинга

# Pipeline анализа

Проект реализует полный цикл обработки данных:

- Сырые данные  
  ↓

  ↓
