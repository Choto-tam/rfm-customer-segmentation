# Notebooks

В данной папке находятся Jupyter Notebook, используемые для исследовательского анализа данных (EDA) и разработки модели RFM-сегментации клиентов.

## Содержимое

**rfm_analysis.ipynb** — основной ноутбук, содержащий полный цикл анализа:

* загрузка данных

* очистка данных

* создание признаков

* агрегация данных по клиентам

* расчет RFM-метрик

* анализ сегментов клиентов

## Назначение

Notebook используется для:

* первичного анализа данных

* проверки гипотез

* разработки и тестирования преобразований

## Результат

В процессе выполнения ноутбука формируются:

* очищенный датасет

* агрегированные данные по клиентам

* RFM-метрики (Recency, Frequency, Monetary)

* RFM-оценки для сегментации клиентов
