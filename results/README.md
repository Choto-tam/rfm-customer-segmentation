# Results

В данной папке находятся результаты анализа данных и RFM-сегментации клиентов.

## Содержимое

* **rfm_table.csv** — итоговая таблица с RFM-метриками и сегментацией клиентов

## Описание результата

Файл содержит агрегированные данные по каждому клиенту:

|Поле|	Описание|
|---|---|
|CustomerID|	идентификатор клиента|
|num_orders| количество заказов|
|total_revenue|	общая выручка|
|total_items|	количество товаров|
|first_purchase|	первая покупка|
|last_purchase|	последняя покупка|
|recency|	давность покупки|
|frequency|	частота|
|monetary|	денежная ценность|
|R_score|	оценка по Recency|
|F_score|	оценка по Frequency|
|M_score|	оценка по Monetary|
|RFM_score|	итоговый RFM-сегмент|

## Назначение

Файл может использоваться для:

* сегментации клиентов

* маркетинговых кампаний

* анализа поведения пользователей

* построения моделей машинного обучения
