# Data

В данной папке находятся исходные данные, используемые в проекте.

## Содержимое

Online Retail.xlsx — транзакционные данные интернет-магазина

## Описание данных

Датасет содержит информацию о покупках:

|Поле |	Описание |
|---|---|
|InvoiceNo|	номер заказа|
|StockCode|	код товара|
|Description	|описание товара|
|Quantity|	количество|
|InvoiceDate|	дата заказа|
|UnitPrice	|цена за единицу|
|CustomerID|	идентификатор клиента|
|Country|	страна|

## Особенности данных

В исходных данных присутствуют:

* пропущенные значения (CustomerID)

* возвраты товаров (отрицательное Quantity)

* дубликаты

* шум и выбросы

Эти проблемы устраняются на этапе очистки данных.
